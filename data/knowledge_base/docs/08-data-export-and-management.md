# Data Export & Management

Data export is available on **all tiers**. Manage exports and data settings under
**Settings > Data Management**.

## Requesting a full data export
Go to **Settings > Data Management > Export** and start an export request. Processing takes up to
**24 hours**. When ready, the export is delivered as a **ZIP file via email** to the requesting
admin. The ZIP contains **CSV files** — one per record type (contacts, deals, activities, notes,
custom fields).

## What's included
A full export includes your contacts, deals and pipeline history, activity/timeline records,
notes, tags, and custom-field values. Attachments are included as a separate folder inside the
ZIP when present.

## Exporting a single report
For just one report's data, use the **Export** button on that report (CSV) — this is immediate
and does not require the 24-hour full-export process. See **Reporting & Analytics**.

## Data retention
Records you delete are retained in a recoverable state for 30 days before permanent deletion, so
an accidental delete can be restored from **Settings > Data Management > Recently Deleted** within
that window.

## GDPR and deletion requests
For GDPR data-subject requests — including the right to erasure / permanent deletion of an
individual's personal data — submit the request and it is routed to EggCRM's **compliance team**.
Support agents do not process GDPR deletion requests directly; these are escalated to compliance.

## Import
Importing data (the inbound counterpart) is done per record type — see **Contacts > Import** in
the **Contact Management** guide for the CSV import flow and de-duplication behavior.
