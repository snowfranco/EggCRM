# Data Export Troubleshooting

## Where exports come from
Full-account exports are requested under **Settings > Data Management > Export** and are available
on **all tiers**. Processing takes **up to 24 hours**, after which the export arrives as a **ZIP
file (containing CSV files) by email** to the requesting admin. See **Data Export & Management**.

## "My export hasn't arrived"
1. **Has 24 hours passed?** Processing can take up to a full day; large accounts take longer.
2. **Check the requesting admin's email**, including spam — the ZIP is delivered by email, not
   shown in-app.
3. **Confirm who requested it** — the link goes to the admin who started the export, not
   necessarily the person asking.
4. If more than 24 hours have passed with no email, re-request the export; if it fails again,
   file a support ticket with the account and request time.

## "The download link expired"
Export download links expire for security after a set window. Re-request the export from
**Settings > Data Management > Export** to generate a fresh link.

## "The ZIP is missing data / wrong format"
A full export contains one **CSV per record type** (contacts, deals, activities, notes,
custom fields), plus an attachments folder when attachments exist. If a record type looks empty,
confirm the account actually has those records and that the requester's role has access to them.
The format is fixed (ZIP of CSVs); for a single report in CSV, use that report's **Export** button
instead (immediate, no 24-hour wait).

## GDPR / deletion requests
A request to **delete** an individual's personal data (right to erasure) is **not** a normal
export — it is routed to the **compliance team**. Agents do not process GDPR deletion directly;
escalate to compliance.

## Recovering deleted records
Deleted records are recoverable for **30 days** under **Settings > Data Management > Recently
Deleted** before permanent deletion.
